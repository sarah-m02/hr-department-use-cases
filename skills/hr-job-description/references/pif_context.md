# PIF Glossary — Strict, Sourced

**This file is a GLOSSARY, not a description.** Every term below has been
verified against a page hosted on `pif.gov.sa` (via a `site:pif.gov.sa` search
that returned the exact wording). Terms that could not be verified from
PIF's own site are **not in this file**. The `hr-job-description` skill
must treat this glossary as literal:

- If a PIF-specific term is needed and it is not in this glossary → do NOT
  invent one; ask the user to provide it, or omit the reference.
- Every entry is annotated with the source URL on `pif.gov.sa`.
- Verbatim wording matters — do not rephrase capitalized names or program
  titles, even slightly.

Verification date: 2026-07 (via `site:pif.gov.sa` search snippets; direct
WebFetch on `pif.gov.sa` returned 403 and could not be used).

## 2. Investment portfolios — the CURRENT three-portfolio structure (verified)

Under PIF's 2026-2030 strategy, PIF's investments are organized into **three
portfolios**. This is the current structure — the older six-pool taxonomy
(Saudi Equity Holdings, Saudi Sector Development, etc.) is HISTORICAL and
must NOT be used in a 2026 JD.

Source: pif.gov.sa/en/our-investments/investment-portfolios/

| Portfolio name (verbatim) | PIF's own description (verbatim, may be paraphrased in a JD) |
|---|---|
| **Vision Portfolio** | *"Catalyzes development of six ecosystems within the local economy… aims to leverage synergies across strategic sectors, maximize value for PIF portfolio companies, and continue to drive the growth of the local economy."* |
| **Strategic Portfolio** | *"Actively manages key strategic assets, maximizing opportunities, building strategic partnerships, and scaling national champions to connect them with global markets."* |
| **Financial Portfolio** | *"Maximizes risk-adjusted returns through thematic investments, delivering resilient value and building strategic international relationships that expand access to global opportunities."* |

## 3. The six ecosystems under the Vision Portfolio (verified verbatim)

Source: pif.gov.sa/en/our-investments/investment-portfolios/ and press-release
announcing the 2026-2030 strategy.

1. **Tourism, Travel & Entertainment**
2. **Urban Development & Livability**
3. **Advanced Manufacturing & Innovation**
4. **Industrials & Logistics**
5. **Clean Energy, Water & Renewables Infrastructure**
6. **NEOM**

Do NOT reorder, rename, or abbreviate these — they are verbatim from PIF's
site. In particular, do NOT write "Tourism and Entertainment," "Urban
Development," or "Clean Energy" as shorthand.

horizontal function department names.** For any role that sits in a
horizontal function, the skill should ask the user to type the specific
team name via the AskUserQuestion "Other" fallback and use whatever the
user provides verbatim.

## 5. Governance and leadership roles (verified)

Titles that appear in PIF press releases:

- **Chairman of the Board** (currently HRH Crown Prince) — do NOT name in a JD body
- **Governor** — do NOT name in a JD body
- **Deputy Governor** (two roles created in 2021)
- **Chief of Staff**
- **Secretary General to the PIF Board of Directors**
- **Board of Directors**
- **Head of [Division]** — e.g. "Head of Local Real Estate Investment Division"

Reports-to language in a JD may reference "Head of [Division]" or the
Governor's office, but must never name individuals.

## 8. Divisions the skill MUST NOT invent

- Do NOT list "Human Capital," "Strategy and Insights," "Portfolio Management,"
  "Corporate Communications," "ESG and Sustainability," "Capital Markets
  Programme," "Local Content and Industrial Procurement," or any other
  horizontal function as a formal PIF department name — none are verified
  against PIF's own site as of this glossary's compilation.
- Do NOT reference the historical six-pool taxonomy ("Saudi Equity Holdings,"
  "Saudi Sector Development," "Saudi Real Estate and Infrastructure," "Saudi
  Giga-Projects" as a pool, "International Strategic Investments," "Diversified
  Global Investments Portfolio," "Treasury Portfolio") in any 2026+ JD. If
  you find yourself needing one, you're using an outdated framing.
- Do NOT use the word "pool" — PIF's own word is **"portfolio."**

## 9. Hard-content prohibitions (from the user, restated here)

The JD MUST NOT contain:

- Any specific compensation figure, range, bonus structure, allowance amount,
  or benefit dollar value
- Any Vision 2030 target percentage or year (e.g. "70% by 2030," "50% women")
- The name of any Saudi national program (Saudization, Nitaqat, azm, HRDF,
  etc.)
- Named individuals (Governor, Chairman, Deputy Governor, division heads)
  in the JD body itself
- Any specific AUM figure or exact headcount number (soft language like
  "one of the world's largest sovereign investors" is fine)
- Any mention of Arabic language as required, preferred, or optional —
  Arabic handling is under separate testing; JD stays silent on it
- Any "Recommended" label on an MCQ option — ordering is signal enough
- Any invented department name (see §8)

## 10. Smart-Division MCQ logic (for the skill)

Given a role title, the skill's Question E options are built from this
glossary. The four options are picked in this order of preference and shown
without any "Recommended" tag:

**A. Role title contains a Vision-Portfolio ecosystem cue:**

| Trigger keyword | Options (in order) |
|---|---|
| "Real Estate," "Property," "Urban," "Housing" | Local Real Estate Investment Division · Vision Portfolio · Strategic Portfolio · Other |
| "Tourism," "Entertainment," "Hospitality," "Travel" | Vision Portfolio (Tourism, Travel & Entertainment) · Strategic Portfolio · Financial Portfolio · Other |
| "Manufacturing," "Innovation" | Vision Portfolio (Advanced Manufacturing & Innovation) · Strategic Portfolio · Financial Portfolio · Other |
| "Industrials," "Logistics," "Supply Chain" | Vision Portfolio (Industrials & Logistics) · Strategic Portfolio · Financial Portfolio · Other |
| "Clean Energy," "Renewables," "Water," "Green" | Vision Portfolio (Clean Energy, Water & Renewables Infrastructure) · Strategic Portfolio · Financial Portfolio · Other |
| "NEOM" | Vision Portfolio (NEOM) · Local Real Estate Investment Division · Strategic Portfolio · Other |

**B. Role title contains a geography or strategic-portfolio cue:**

| Trigger keyword | Options (in order) |
|---|---|
| "International," "Global," "Foreign" (investment role) | International Investments Division · Strategic Portfolio · Financial Portfolio · Other |
| "MENA," "Regional" (investment role) | MENA Investments Division · Strategic Portfolio · Vision Portfolio · Other |
| "Public Markets," "Listed Equities," "Fixed Income," "Thematic" | Financial Portfolio · Strategic Portfolio · Vision Portfolio · Other |
| "National Champion," "Portfolio Company Value Creation," "Strategic Asset" | Strategic Portfolio · Vision Portfolio · Financial Portfolio · Other |

**C. Role title cues a horizontal / corporate function:**

Any keyword suggesting Legal, Finance, HR, Comms, Risk, Compliance, Audit,
Technology, Strategy, ESG, Procurement, or similar:

Options: `Corporate function or specialist team (please specify) · Vision Portfolio · Strategic Portfolio · Other`

If the user picks the first option, use the AskUserQuestion "Other" free-text
fallback to capture the team name **verbatim** and use exactly that string
in the JD's Work Details Block and Role Overview. Do NOT map it to a
guessed department name.

**D. Fallback (no keyword rule fires):**

`Strategic Portfolio · Vision Portfolio · Financial Portfolio · Other`

## 11. Hard rules the skill MUST obey when using this glossary

1. Never emit any of the items on the §7 forbidden list.
2. Never invent a corporate department name (§8). If in doubt, ask the user
   or omit the reference.
3. Never use the six-pool historical taxonomy in a 2026+ JD.
4. Never use the word "pool" — PIF's own word is "portfolio."
5. Never use "Recommended" as an MCQ label.
6. Never assert Arabic as a required, preferred, or optional language.
7. Never name individuals (Governor, Chairman, division heads) in the JD body.
8. Never quote a specific AUM figure or exact headcount number in a JD.
9. Verbatim ecosystem names in §3 must be preserved exactly — no reordering,
   renaming, or shortening.
10. Every PIF-specific claim in a JD should trace to an entry in this
    glossary. If it doesn't, question it.
